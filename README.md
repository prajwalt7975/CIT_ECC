# 7-bit ALU ASIC Design

## Overview
This project implements a 7-bit Arithmetic Logic Unit (ALU) using Verilog HDL for ASIC design flow.

## Features
- Addition
- Subtraction
- AND
- OR
- XOR

## Modules
- tt_um_alu7b.v
- serial_alu_ctrl.v
- alu_7b.v

## Tools Used
- Verilog HDL
- OpenLane
- OpenROAD
- GitHub
